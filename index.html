1. Information We Collect

Data You Provide

  • Text Input: Any English text you enter for translation is processed locally within your browser and is never sent to external servers.
  • Learned Vocabulary: Words learned during your session are stored temporarily in your browser's memory and are automatically deleted when you close the application.
  • Translation History: Your translation script (if generated) exists only in session memory and is not permanently stored.

Audio/Speech Synthesis

‚ö†Ô∏è Important: Speech Synthesis Permissions

The Aetheliel Translator uses your browser's built-in Web Speech Synthesis API (specifically attempting to use "Microsoft Sonia Desktop" voice on Windows) for text-to-speech functionality. This feature:

  • Does NOT record your voice or audio
  • Does NOT access your microphone for input
  • Only produces speech output (computer-generated voice)
  • Can be toggled on/off at any time using the "speak" command or toggle button
  • Uses your operating system's native speech synthesis engine

Note: Your browser or operating system may request microphone permissions as part of the Speech API initialization. This is a standard browser security feature. The Aetheliel Translator does NOT record, access, or transmit any audio from your microphone.

2. How We Use Your Information

All data processing occurs exclusively in your browser:

  • Translation Engine: English text is converted to Aetheliel using JavaScript algorithms running locally in your browser
  • Script Generation: Aethrasil script (Cherokee-inspired glyphs) is generated client-side
  • Morphological Processing: Grammar application and compound word formation happen entirely in your browser's memory
  • Text-to-Speech: Your browser's or operating system's native speech synthesis reads Aetheliel aloud (if enabled)
  • Vocabulary Learning: New words are stored in volatile browser memory for the current session only
  • Script File Generation: Translation scripts are created in session memory and can be downloaded by you manually

3. Data Storage and Retention

Session-Only Memory Storage

The Aetheliel Translator stores vocabulary and learned words exclusively in your browser's RAM (volatile memory) during your active session. This data:

  • Exists only while the application is running in your browser tab
  • Is immediately and permanently deleted when you close the browser tab or window
  • Is never written to disk, permanent storage, or any database
  • Is not accessible to other websites, applications, or browser tabs
  • Cannot be recovered after the session ends

No Persistent Storage Mechanisms

The Aetheliel Translator deliberately does NOT use any of the following:

  • HTTP Cookies
  • LocalStorage API
  • SessionStorage API
  • IndexedDB
  • Web SQL
  • Service Workers with caching
  • Any other browser storage mechanism

Note: All learned vocabulary and translation history exists only in your current session and is permanently deleted when you close the app.

4. Third-Party Services and Dependencies

The Aetheliel Translator web application is a completely standalone, self-contained application that does NOT use:

  • External APIs or web services
  • Analytics services (Google Analytics, etc.)
  • Advertising networks
  • Social media integrations
  • Content Delivery Networks (CDNs) for libraries
  • Remote fonts or resources

Browser-Provided Speech Synthesis

The only external dependency is your web browser's built-in Speech Synthesis API, which is provided by:

  • Chrome/Edge (Windows): Uses Microsoft Speech Platform voices according to Microsoft's privacy policy
  • Chrome/Edge (macOS): Uses macOS system voices according to Apple's privacy policy
  • Firefox: Uses operating system voices according to Mozilla's and your OS vendor's privacy policies
  • Safari: Uses macOS/iOS system voices according to Apple's privacy policy

Speech synthesis is handled entirely by your browser and operating system. The Aetheliel Translator has no control over, visibility into, or access to this process. We recommend reviewing your browser's and operating system's privacy policies for details on how speech synthesis is handled.

5. Your Rights and Controls

You maintain complete control over the application and your data:

  • Toggle Speech: Type "speak" command or use the toggle button to enable/disable text-to-speech at any time

  • View Learned Words: Type "export vocab" to see all words learned in the current session
  • End Session: Simply close the browser tab to permanently delete all session data
  • Revoke Permissions: You can revoke any browser permissions (such as microphone access) through your browser settings at any time
